# ContractIQ — Promptathon 2026 POC

A demo app showing what a student team could build at Promptathon 2026.

## What it does
- Takes any contract text as input
- Uses Gemini AI to extract all obligations, parties, deadlines, and risk levels
- Shows a structured output ready for a compliance team
- Estimates cost and time savings for the business case

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
